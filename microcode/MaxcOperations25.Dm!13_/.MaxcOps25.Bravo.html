25.2. Changing and Printing this Document
Three dump files are stored on [IVY]<Maxc>. They are: MaxcOperations0-24Bravo.Dm, MaxcOperations0-24Press.Dm and MaxcOperations25.Dm. The numbers 0-24 and 25 refer to sections of the document. Contained in these dump files are all files necessary to update this document, as well as command files to create the dump files and to print the document in two parts; sections 0-24 and section 25. Below is a list of the contents of each command file and information concerning the amount of disk space required on an Alto to load the dump files and run the command files associated with them.
To make changes to sections 0-24, load dump file MaxcOperations0-24Bravo.Dm and make any necessary changes to the bravo files contained therein. Create a press file for each bravo file that is changed. When finished, run command file SaveMaxcOperations0-25Bravo.Cm to store the changed files on Ivy. Now load the dump file MaxcOperations0-24Press.Dm (before starting the load delete unnecessary files to make room. You will need approximately 1500 - number of pages used by the MaxcOpsX.Press files remaining from the pervious session). Skip [OldFile]s as they occur during the transfer. Run SaveMaxcOperations0-24Press.Cm to store the changed press files on Ivy. Run PrintMaxcOperations0-24.Cm to print sections 0-24 and store the file MaxcOperations0-24.Press on Ivy.
Making changes to section 25 is a little different in that changes to the Figures requires the use of PressEdit as described on page 105, and when running PrintMaxcOperations25.Cm you must use Bravo to create MaxcOps25.Press, otherwise follow the procedure as described above using the appropriate command files.
